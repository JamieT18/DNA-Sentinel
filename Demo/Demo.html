DNA Sentinel

DNA Sequencing Checker

Upload a plain text file containing a DNA sequence to verify its integrity. Our system performs multiple security checks:

  1. MIME Type Verification: The file must be a plain text file (.txt).
  2. Letter Validation: The sequence must contain only valid nucleotides (A, T, C, and G).
  3. Binary Conversion & Analysis: The sequence is converted into binary and scanned for malicious patterns ("1010101010").
Accepted File Type: .txt
Note: Additional file formats may be permitted based on company policy.

Example files for testing:
- Safe sample: ATCGATCGATCG
- Letter error sample: ATCGXTGATCG (contains an invalid character "X")
- Binary error sample: CCCCC (each "C" converts to "10", producing "1010101010")
